# Hospital Management CRUD Project (PHP + MySQL)

This is another assignment project where I made a CRUD application for Hospital Patients using PHP and MySQL. The app can add, view, update and delete patient records.

## What I used
- XAMPP (Apache + MySQL)
- PHP
- MySQL
- HTML/CSS

## Files in this project
- database.sql - creates the database and patients table
- db.php - connects to the database
- index.php - shows all patients (main page)
- create.php - form to add a new patient
- edit.php - form to update a patient
- delete.php - deletes a patient

## How to run it

1. Install XAMPP and start Apache + MySQL.
2. Put this folder inside htdocs.
3. Open phpMyAdmin, go to SQL tab, paste the content of database.sql and run it. This will create the database and table.
4. Open browser and go to: localhost/hospital-crud/index.php
5. From here you can add, edit and delete patient records.

## Notes
- This is a CRUD (Create, Read, Update, Delete) assignment.
- I picked the Patients table and built all CRUD operations on it as required.
